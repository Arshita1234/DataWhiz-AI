\# 🚀 DataWhiz Pro: Agentic PDF Insights



A high-performance RAG (Retrieval-Augmented Generation) system built with \*\*Streamlit\*\*, \*\*LangChain\*\*, and \*\*Google Gemini 3 Flash\*\*.



\### ✨ Key Features

\* \*\*Multi-Doc Support:\*\* Upload and query multiple PDFs simultaneously.

\* \*\*Persistent Knowledge:\*\* Uses ChromaDB to save embeddings locally across sessions.

\* \*\*Smart Context:\*\* Powered by the latest 2026 `gemini-embedding-001` model.

\* \*\*Interactive Chat:\*\* Full chat history tracking for seamless exploration.



\### 🛠️ Installation

1\. Clone the repo: `git clone https://github.com/Arshita1234/DataWhiz-AI.git`

2\. Install dependencies: `pip install -r requirements.txt`

3\. Run the app: `streamlit run app.py`
